# fetch-rs
## About
A System Information Tool that i wrote to learn rust.

<p align="center"><img src="https://github.com/Mhirii/fetch-rs/blob/main/assets/screenshot.png" width="80%"></p>

## Usage
```bash
fetch-rs [options]
```

>  Options:  
>     --os  
>     --host  
>     --kernel  
>     --user  
>     --uptime  
>     --cpu  
>     --memory  
>     --packages  
>     --wm  
>     --session  
>     --terminal  
>     --logo  
> -h, --help      Print help  
> -V, --version   Print version
